
# Setting up the project locally

## Pre Build commands
``` bash

# Installing Node js
- nvm install 14

# Installing all the dependencies of project
- yarn

# Installing Bootstrap
- yarn bootstrap

# Change Directory to packages
- cd packages

# Change Directory to common
- cd common
```


## build commands
``` bash
- yarn prepare
- cd ..
- cd web
- yarn prestart
- yarn build
- yarn export

```


But these are instructions to run locally
